# Python Custom Warehouse Management System (WMS)

## Descrizione del Progetto
Questo progetto consiste nello sviluppo di un sistema gestionale custom in Python per l'automazione delle operazioni di magazzino e vendita di un'attività commerciale (negozio di prodotti biologici/vegani). 

L'obiettivo principale è stato quello di creare uno strumento robusto in grado di gestire il ciclo di vita del prodotto: dall'approvvigionamento alla vendita, con persistenza dei dati su file system e generazione automatica di reportistica per il management.

## Competenze Tecniche Dimostrate
* **Programmazione Procedurale e Funzionale:** Implementazione di una logica di business modulare e riutilizzabile.
* **Data Persistence (I/O):** Gestione della lettura e scrittura dei dati tramite file CSV per simulare un database leggero.
* **Gestione delle Eccezioni:** Implementazione di controlli per garantire l'integrità dei dati (es. controllo disponibilità scorte, input utente validati).
* **User Story Mapping:** Approccio metodologico nella fase di design per tradurre i requisiti dell'utente in funzionalità tecniche "aggredibili".

## Funzionalità Principali
- **Gestione Inventario:** Caricamento, aggiornamento e consultazione dello stock in tempo reale.
- **Ciclo di Vendita:** Registrazione delle transazioni con aggiornamento automatico delle giacenze.
- **Reporting & Analytics:** Funzione di visualizzazione del magazzino e calcolo automatico dei profitti basato sui margini di vendita.
- **Persistenza:** Salvataggio automatico dello stato del magazzino per garantire la continuità dei dati tra diverse sessioni.

## Struttura del Repository
- `Gestionale prodotti vegani.ipynb`: Notebook principale contenente il codice sorgente commentato e la documentazione delle scelte progettuali.
- `magazzino.csv`: Database flat-file per la memorizzazione dei prodotti (generato a runtime).
